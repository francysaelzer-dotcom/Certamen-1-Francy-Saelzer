# Portafolio Profesional

# Tecnologías Utilizadas

- HTML5: Estructura semántica de la página.
- CSS3: Diseño visual, implementando Flexbox y CSS Grid.

# Estructura del Proyecto

- `index.html`: Contiene toda la estructura y contenido del portafolio.
- `estilos.css`: Contiene los estilos de la página, incluyendo diseño responsivo para móviles y tablets.

# Requisitos Cumplidos
- Uso de etiquetas semánticas (`header`, `nav`, `main`, `section`, etc.).
- Navegación interna (anclas).
- Diseño responsivo con Media Queries.
- Uso de Flexbox y CSS Grid.
- Formulario de contacto y tabla de habilidades.
